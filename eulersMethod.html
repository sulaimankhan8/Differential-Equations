Eulers Method



Euler's Method is a straightforward numerical approach for solving ordinary differential equations (ODEs) with an initial value. It approximates the solution by taking small steps from an initial condition and using the derivative information.


The General Approach


Given a differential equation of the form:

dy dx=  f(x,y)

with an initial conditionyy(x0)= y0,Euler's method approximates the solution at discrete points.

Steps of Euler's Method


1.Start at the initial point:(x0,y0)

2.Calculate the slope using the differential equation: evaluate the derivatives at the initial point f(x0,y0)

3.Move to the next point:

x(n+1)= xn+h


y(n+1)=yn+ h * f(xn,yn)


wherehis the step size. to be reach x

4.Repeat steps 2 and 3 until the desired x-value is reached



Example

Consider the differential equation:

dy dx=  x+y

with the initial condition y(0) = 1


To approximate y at x= 1 using a step size hsuch that i = 5, where n is numberof steps .

Given:

  • Differential equation: dy dx=  x+y

  • initial condition: y(0)= 1

  • Number of steps: n= 5

  • Find yat x= 1

    Step-by-Step Calculation

  • First, we determine the step size h:


    h = x-x0 n = 1-0 5= 0.2

  • Using Euler's Method, we iteratively calculate y for each steps:

      initial values x0= 0, y0= 1

    1. Iteration 1:

      y1= y0 + h * f(x0, y0)


      y1=1+0.2*(0+1)=1+0.2=1.2


      x1= x0+ h = 0 + 0.2=0.2


    2. Iteration 2:

      y2 = y1 + h * f(x1, y1)

      y2 = 1.2 + 0.2 * (0.2 + 1.2) = 1.2 + 0.2 * 1.4 = 1.2 + 0.28 = 1.48

      x2 = x1 + h = 0.2 + 0.2 = 0.4


    3. Iteration 3:

      y3 = y2 + h * f(x2, y2)

      y3 = 1.48 + 0.2 * (0.4 + 1.48) = 1.48 + 0.2 * 1.88 = 1.48 + 0.376 = 1.856

      x3 = x2 + h = 0.4 + 0.2 = 0.6


    4. Iteration 4:

      y4 = y3 + h * f(x3, y3)

      y4 = 1.856 + 0.2 * (0.6 + 1.856) = 1.856 + 0.2 * 2.456 = 1.856 + 0.4912 = 2.3472

      x4 = x3 + h = 0.6 + 0.2 = 0.8


    5. Iteration 5:

      y5 = y4 + h * f(x4, y4)

      y5 = 2.3472 + 0.2 * (0.8 + 2.3472) = 2.3472 + 0.2 * 3.1472 = 2.3472 + 0.62944 = 2.97664

      x5 = x4 + h = 0.8 + 0.2 = 1.0
